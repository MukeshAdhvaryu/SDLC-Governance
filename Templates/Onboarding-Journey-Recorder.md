# Onboarding Journey Recorder

> One copy of this document is created per new joiner. It is a living record - not
> a form to be filed. Update it as access is granted, inductions are completed, and
> training is done. It is the shared source of truth between the new joiner, their
> manager, their buddy, HR, and IT.
>
> Status symbols: [x] Done - [ ] Not started - [*] In progress / requested - [!] Blocked - [.] Deferred

## Day 1 Logistics

Completed by HR or the hiring manager and sent to the new joiner before their first day.
Be specific. Generic guidance ("take public transport") is not useful.

| Topic | Details |
|-------|---------|
| Office address | Full postal address with postcode |
| Who to ask for at reception | Name and contact number |
| What to bring on day one | Passport / ID for HR admin, laptop (if bring-your-own), etc. |
| Building entry | Where to enter, what to say, where to wait |
| Nearest train station(s) | Station names and approximate walk time |
| Nearest bus stop(s) | Route numbers and stop names |
| Nearest car parking | Location, cost, any corporate discount code |
| Recommended hotels (if travelling) | Hotel names, distance to office, any corporate rate or discount code |
| Dress code | Formal / smart-casual / casual |
| Start time | Confirm the arrival time (not just the official start time) |
| Manager's mobile number | For use if running late or lost |
| Lunch | Whether the team eats together on day one, nearest food options, any canteen details |
| Any access card or temp pass logistics | What to do if pass is not ready |

## Journey Log

Append dated notes as the onboarding progresses. Do not edit past entries.
This is a chronological record, not a curated summary.

```
YYYY-MM-DD - Journey Recorder created. Laptop ordered. Email account requested.
YYYY-MM-DD - Laptop delivered and provisioned. Email confirmed working.
YYYY-MM-DD - Day one. Building access working. Repo access requested (ref: INC-12345).
YYYY-MM-DD - Repo access granted. VPN setup complete. Standup attended.
YYYY-MM-DD - Service portal request for [tool name] escalated - no response at 5-day mark.
YYYY-MM-DD - 30-day checkpoint completed. All Tier 2 access granted. Bespoke software still pending.
YYYY-MM-DD - 60-day checkpoint. Fully contributing. Buddy relationship formally concluded.
```
